# 🚀 Hacktiviti - Site Vitrine

🌐 **Démo en ligne :**
https://valpedro59.github.io/site-vitrine-hacktiviti/

---

## 📖 À propos

Hacktiviti accompagne les entreprises, les entrepreneurs et les organisations dans leur transformation digitale.

Notre objectif est de concevoir des solutions web modernes, performantes et intuitives qui permettent à nos clients d'améliorer leur visibilité, développer leur activité et renforcer leur image de marque.

## 🚀 Installation

Clonez le dépôt :

```bash
git clone https://github.com/ValPedro59/site-vitrine-hacktiviti.git
```

Accédez au dossier :

```bash
cd site-vitrine-hacktiviti
```

Ouvrez simplement le fichier :

```text
index.html
```

ou utilisez un serveur local comme **Live Server** dans Visual Studio Code.
