# 🚀 Hacktiviti - Leader du Numérique au Congo

Hacktiviti est une plateforme web moderne et hautement performante conçue pour une agence de solutions digitales basée à Pointe-Noire, République du Congo. Ce projet vitrine met en avant les expertises de l'agence à travers un design épuré, réactif et axé sur l'accessibilité.

---

## 💎 Caractéristiques du Projet

- **Architecture 100% HTML5 & CSS3 Pur :** Aucun framework lourd (ni Bootstrap, ni Tailwind) et aucun script JavaScript. Le site est ultra-léger et se charge instantanément, même avec une connectivité réduite.
- **Approche Mobile-First :** Le design a été entièrement pensé et optimisé pour les téléphones portables avant d'être déployé sur les tablettes et ordinateurs de bureau.
- **Typographie Fluide (`CSS clamp()`) :** Les tailles du titre principal de la section hero, des paragraphes et des titres de section s'adaptent de manière parfaitement fluide à la largeur de l'écran, éliminant les sauts de texte brusques.
- **Menu Burger sans JS (Checkbox Hack) :** Le système d'ouverture, de fermeture et d'animation en croix (X) du menu de navigation mobile repose exclusivement sur les fonctionnalités CSS avancées (`:checked` et sélecteurs frères `~`).
- **Graphismes vectoriels légers (SVGs Inline) :** Toutes les icônes de la grille de services et de la section contact utilisent le format SVG intégré au HTML. Ils sont stylisés dynamiquement en CSS avec la propriété `currentColor`.

## 🎨 Palette de Couleurs & Identité Visuelle

Le projet s'appuie sur une palette de variables CSS rigoureusement sélectionnées pour garantir un contraste maximal et un confort visuel optimal (normes WCAG d'accessibilité) :

| Variable CSS     | Code Couleur | Utilisation Principale                                                |
| :--------------- | :----------- | :-------------------------------------------------------------------- |
| `--bg-primary`   | `#155dfc`    | Bleu Électrique (Actions principales, Titres H2, Éléments clés)       |
| `--bg-secondary` | `#8f8cb8`    | Violet Pastel (Sous-titres discrets sur fond sombre)                  |
| `--accent-color` | `#4ebe9e`    | Vert Menthe (Touches de validation, puces, contrastes ciblés)         |
| `--text-primary` | `#faf7ff`    | Blanc Cassé (Fond de header, textes sur arrières-plans sombres)       |
| `--text-black`   | `#1b1a28`    | Noir Foncé Bleuté (Corps de texte sur fond blanc, confort de lecture) |

## 🛠️ Installation et Utilisation Locale

Étant donné que ce projet est développé en technologies web natives pures, il ne nécessite aucun gestionnaire de paquets (`npm`) ni aucune compilation.

1. **Cloner le projet** ou télécharger les fichiers sources sur votre machine.
2. **Ouvrir le fichier** `index.html` directement dans n'importe quel navigateur web moderne (Chrome, Firefox, Safari, Edge).
3. Pour le développement, l'utilisation d'une extension de serveur local (comme _Live Server_ sur VS Code) est recommandée pour observer les modifications en temps réel.
